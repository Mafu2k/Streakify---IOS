# Streakify---IOS
Aplikacja iOS do śledzenia nawyków dziennych z funkcją serii i statystyk.

# Opis
Streakify-iOS to aplikacja do budowania nawyków, która zamienia codzienne rutyny w ekscytującą grę z modelem serii. Pomaga utrzymać dyscyplinę przez rywalizację ze znajomymi i globalny ranking , co motywuje do regularności. Daje poczucie dumy z wizualnych postępów i nagradza konsekwencję, budując lepsze nawyki życiowe.


# Funkcjonalności

Rejestracja nawyków z niestandardowymi nazwami, ikonami i celami dziennymi
Wizualizacja serii na interaktywnym kalendarzu
Osobiste statystyki (procent ukończenia, najdłuższa seria, dni aktywne)
Lokalne przechowywanie danych z automatycznym backupem
Rywalizacja z znajomymi przez unikalne kody zaproszeń, globalny ranking 
Powiadomienia push o utraconych streakach
Gesty swipe do szybkiego oznaczania nawyków
Tryb ciemny i jasny z automatycznym przełączaniem
